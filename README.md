




















## Employee Management System using React + Spring Boot (Ongoing Project..)

### Tech Stack
- Spring Boot 3
- Spring Data JPA (Hibernate 6)
- MySQL
- Maven
- Postman
- React Js
- Vite Js
- Bootstrap CSS
- npm
- Axios

### CRUD REST APIs (Backend)

Requirements:
- Add Employee
- Get Employee
- Get All Employee
- Update Employee
- Delete Employee

### React App (Frontend)

Requirments:

User should able perform below CRUD operations
- Add New Employee
- List All Employees
- Update Existing Employee
- Delete Existing Employee
